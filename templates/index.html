About the Project

This project is developed as part of CENG534 - Deep Learning for Natural Language Processing course. It aggregates news from multiple international sources including:

  • CNN (English)
  • El País (Spanish)
  • Le Monde (French)
  • Der Spiegel (German)

API Documentation

All endpoints require authentication using the API token in the Authorization header.

Get News by Language

GET /api/news/{language}

Available languages: german, spanish, french, english

Parameters:

  • page: Page number (default: 1)
  • limit: Results per page (default: 10, max: 100)

Search News

GET /api/news/search?q={keyword}

Search across all news sources

Parameters:

  • q: Search keyword (required)
  • page: Page number
  • limit: Results per page

Recent News

GET /api/news/recent

Get most recent news from all sources

Parameters:

  • page: Page number
  • limit: Results per page
